# CV HTML / CSS

Projet de CV en ligne réalisé en HTML et CSS à partir d'une maquette.

## Structure

- `index.html` : page principale du CV
- `assets/css/` : feuilles de style séparées
- `assets/images/` : photo de profil et favicon
- `docs/issues.md` : base de découpage des tâches pour GitHub

## Points pris en compte

- HTML sémantique
- CSS séparé du HTML
- Flexbox et CSS Grid
- largeur cible de 1160 px avec adaptation mobile
- accessibilité de base : liens explicites, focus visible, texte alternatif, skip link
- SEO de base : balises meta, hiérarchie de titres, contenus structurés

## Personnalisation

Les données actuelles sont fictives. Remplacer le nom, les coordonnées, les expériences, les liens sociaux et la photo par vos informations avant la remise finale.
